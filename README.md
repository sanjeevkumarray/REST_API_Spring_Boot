Spring Boot REST API
A simple Spring Boot REST API for managing books.

Endpoints
GET /books: Retrieve a list of books.
POST /books: Create a new book.
PUT /books/{bookId}: Update a book.
DELETE /books/{bookId}: Delete a book.
